## Types

### strview
```c
typedef struct stv_strview_t strview;
struct stv_strview_t {
    const char* data;
    size_t      len;
};
```
The string view type. Represents a read-only fragment of a string, not null-terminated, and does not own the data.

**Members:**
- `data` : Pointer to the first character.
- `len` : Number of bytes in the view.

### stv_charClassFn
```c
typedef int (*stv_charClassFn)(int);
```
Function pointer type for character classification. Should behave like the standard `is*` functions (e.g., `isdigit`, `isspace`) – returns non-zero if the character belongs to the class, zero otherwise.

### stv_forEachFn
```c
typedef void (*stv_forEachFn)(char ch, size_t idx, strview ctx);
```
Callback function pointer type for [`stv_forEach`](#stv_foreach).
- `ch` : Current character.
- `idx` : Zero-based index of the character.
- `ctx` : The original `strview` passed to the iteration.

### stv_cstrOptions
```c
typedef enum {
    stv_Default = 0,  // 0b00000000
    stv_ToUpper = 1,  // 0b00000001
    stv_ToLower = 2,  // 0b00000010
    stv_Reverse = 4,  // 0b00000100
} stv_cstrOptions;
```
Bitwise options for `stv_opt_cstr` to control output transformations.

**Values:**
- `stv_Default` (0): No transformation.
- `stv_ToUpper` (1): Convert characters to uppercase using an ASCII mask.
- `stv_ToLower` (2): Convert characters to lowercase using an ASCII mask.
- `stv_Reverse` (4): Reverse the character order.

Multiple options can be combined with `|`. If both `stv_ToUpper` and `stv_ToLower` are set, the result is swapCase (Abc -> aBC).

## Creation

### stv_new
```c
strview stv_new(const char* c_str);
```
Create a view from a C string (up to the null terminator).

**Parameters:**
- `c_str` : Pointer to a null-terminated C string. May be `NULL`.

**Return:**
- A `strview` spanning from `c_str` to the null terminator. Returns an empty view ([`stv_nullstv`](#stv_nullstv)) if `c_str` is `NULL`.

**Details:**
- Equivalent to `stv_create(c_str, '\0', stv_npos)`.

### stv_create
```c
strview stv_create(const char* str, unsigned char endchar, size_t maxlen);
```
Create a view from a contiguous character sequence, stopping at a specified terminator or maximum length.

**Parameters:**
- `str` : Source string pointer. May be `NULL`.
- `endchar` : Terminator character (stop before this character; the character itself is not included).
- `maxlen` : Maximum number of bytes to scan.

**Return:**
- A `strview` starting at `str` and extending up to the first occurrence of `endchar` or `maxlen` bytes, whichever comes first. Returns an empty view if `str` is `NULL`.

### stv_literal
```c
#define stv_literal(str) stv_makestv((str), sizeof(str) - 1)
```
Macro to create a view from a string literal. The length is computed at compile time (`sizeof`).

### stv_makestv
```c
#ifdef __cplusplus
    #define stv_makestv(data_v, len_v) (strview{/* .data= */ (data_v), /* .len= */ (len_v)})
#else
    #define stv_makestv(data_v, len_v) ((strview){.data = (data_v), .len = (len_v)})
#endif

```
Macro to construct a `strview` from a pointer and a length.

### stv_nullstv
```c
#define stv_nullstv stv_makestv(nullptr, 0)
```
Predefined empty view with `data = nullptr` and `len = 0`.

## Slicing

### stv_slice
```c
strview stv_slice(strview stv, size_t begin_pos, size_t end_pos);
```
Extract a sub-view covering the range `[begin_pos, end_pos)`.

**Parameters:**
- `stv` : Source view.
- `begin_pos` : Start index (inclusive).
- `end_pos` : End index (exclusive). Use [`stv_end`](#stv_end) to indicate up to the end.

**Return:**
- A sub-view if the range is valid; otherwise an empty view.

**Details:**
- If `end_pos` is `stv_npos` (i.e., `stv_end`), it is treated as `stv.len`.
- Returns an empty view when `begin_pos > end_pos`, `begin_pos > stv.len`, or `data` is `NULL`.

### stv_removeStart
```c
strview stv_removeStart(strview stv, size_t len);
```
Remove `len` bytes from the beginning. Equivalent to `stv_slice(stv, len, stv_end)`.

**Parameters:**
- `stv` : Source view.
- `len` : Number of bytes to remove.

**Return:**
- The view without the first `len` bytes, or an empty view if `len >= stv.len`.

### stv_removeEnd
```c
strview stv_removeEnd(strview stv, size_t len);
```
Remove `len` bytes from the end. Equivalent to `stv_slice(stv, stv_begin, stv.len - len)`.

**Parameters:**
- `stv` : Source view.
- `len` : Number of bytes to remove.

**Return:**
- The view without the last `len` bytes, or an empty view if `len >= stv.len`.

### stv_removePrefix
```c
strview stv_removePrefix(strview stv, strview prefix);
```
If the view starts with `prefix`, remove it; otherwise return unchanged.

**Parameters:**
- `stv` : Source view.
- `prefix` : Prefix to remove. An empty prefix matches and returns the original view.

**Return:**
- The view after the prefix, or the original view if it does not start with the prefix.

### stv_removeSuffix
```c
strview stv_removeSuffix(strview stv, strview suffix);
```
If the view ends with `suffix`, remove it; otherwise return unchanged.

**Parameters:**
- `stv` : Source view.
- `suffix` : Suffix to remove. An empty suffix matches and returns the original view.

**Return:**
- The view without the suffix, or the original view if it does not end with the suffix.

## Splitting

### stv_split
```c
strview stv_split(strview stv, strview sep, strview* remaining);
```
Split at the first occurrence of the separator. The part before the separator is returned, and the remainder is stored in `*remaining` if not `NULL`.

**Parameters:**
- `stv` : View to split.
- `sep` : Separator view. If empty, splits character by character (returns first char, stores rest).
- `remaining` : Optional output pointer for the part after the separator. If `NULL`, the remainder is discarded.

**Return:**
- The part before the separator, or the entire view if the separator is not found.

**Details:**
- When `stv` is empty, returns an empty view and does not modify `*remaining`.

### stv_splitLines
```c
strview stv_splitLines(strview stv, strview* remaining);
```
Split at the first line break (CR, LF, or CRLF). The returned line does not include the break characters.

**Parameters:**
- `stv` : View to split.
- `remaining` : Optional output pointer for the part after the line break. If `NULL`, the remainder is discarded.

**Return:**
- The current line, or the entire view if no line break is found.

**Details:**
- If `stv` is empty, returns an empty view without modifying `*remaining`.
- Supports Windows-style CRLF as a single break.

### stv_splitWords
```c
strview stv_splitWords(strview stv, strview* remaining);
```
Skips leading whitespace and extracts the next contiguous sequence of non‑whitespace characters.

**Parameters:**
- `stv` : View to split.
- `remaining` : Optional output pointer for the part after the word. If `NULL`, the remainder is discarded.

**Return:**
- The current word, or an empty view if no word is found (empty view or all whitespace).

**Details:**
- Whitespace is defined by [`stv_whitespace`](#stv_whitespace).
- When `stv` is empty, `*remaining` is not modified.

### stv_beforeDelim
```c
strview stv_beforeDelim(strview stv, strview delim);
```
Return the portion of the view before the first occurrence of the delimiter.

**Parameters:**
- `stv` : View to examine.
- `delim` : Delimiter view. If empty, returns an empty view.

**Return:**
- The part before the delimiter; entire view if delimiter not found; empty view if `delim` is empty.

### stv_afterDelim
```c
strview stv_afterDelim(strview stv, strview delim);
```
Return the portion of the view after the first occurrence of the delimiter.

**Parameters:**
- `stv` : View to examine.
- `delim` : Delimiter view. If empty, returns the entire view unchanged.

**Return:**
- The part after the delimiter; empty view if delimiter not found.

## Trimming

### Generic trim macros

#### stv_trim
```c
#define stv_trim(stv, target) _Generic((target), strview: stv_trimChs, stv_charClassFn: stv_trimIf)((stv), (target))
```
Dispatches to `stv_trimChs` or `stv_trimIf` based on the type of `target`.  
If `target` is a `strview`, it is treated as a character set; if it is a `stv_charClassFn`, it is used as a classification function.

#### stv_trimStart
```c
#define stv_trimStart(stv, target) _Generic((target), strview: stv_trimStartChs, stv_charClassFn: stv_trimStartIf)((stv), (target))
```
Dispatches to `stv_trimStartChs` or `stv_trimStartIf`. Trims only the start.

#### stv_trimEnd
```c
#define stv_trimEnd(stv, target) _Generic((target), strview: stv_trimEndChs, stv_charClassFn: stv_trimEndIf)((stv), (target))
```
Dispatches to `stv_trimEndChs` or `stv_trimEndIf`. Trims only the end.

### Charset-based trim

#### stv_trimChs
```c
strview stv_trimChs(strview stv, strview charset);
```
Remove any characters belonging to `charset` from both ends.

**Parameters:**
- `stv` : Source view.
- `charset` : Character set view. If empty, the view is returned unchanged.

**Return:**
- The trimmed view.

#### stv_trimStartChs
```c
strview stv_trimStartChs(strview stv, strview charset);
```
Remove characters belonging to `charset` from the beginning only.

**Parameters:**
- `stv` : Source view.
- `charset` : Character set view.

**Return:**
- The trimmed view.

#### stv_trimEndChs
```c
strview stv_trimEndChs(strview stv, strview charset);
```
Remove characters belonging to `charset` from the end only.

**Parameters:**
- `stv` : Source view.
- `charset` : Character set view.

**Return:**
- The trimmed view.

### Classification-function trim

#### stv_trimIf
```c
strview stv_trimIf(strview stv, stv_charClassFn handle);
```
Remove characters from both ends that satisfy the classification function.

**Parameters:**
- `stv` : Source view.
- `handle` : Classification function. If `NULL`, no trimming is performed.

**Return:**
- The trimmed view.

#### stv_trimStartIf
```c
strview stv_trimStartIf(strview stv, stv_charClassFn handle);
```
Like `stv_trimIf`, but only trims the start.

**Parameters:**
- `stv` : Source view.
- `handle` : Classification function.

**Return:**
- The trimmed view.

#### stv_trimEndIf
```c
strview stv_trimEndIf(strview stv, stv_charClassFn handle);
```
Like `stv_trimIf`, but only trims the end.

**Parameters:**
- `stv` : Source view.
- `handle` : Classification function.

**Return:**
- The trimmed view.

## Searching

### stv_search
```c
size_t stv_search(strview stv_text, strview stv_pat);
```
Search for the first occurrence of `stv_pat` in `stv_text`. Automatically chooses the algorithm (naive for short patterns, Sunday otherwise).

**Parameters:**
- `stv_text` : Text to search in.
- `stv_pat` : Pattern to search for.

**Return:**
- Start index of the first match; `0` if pattern is empty; [`stv_npos`](#stv_npos) if not found.

### stv_naiveSearch
```c
size_t stv_naiveSearch(strview stv_text, strview stv_pat);
```
Naive (brute-force) substring search.

**Parameters:**
- `stv_text` : Text to search in.
- `stv_pat` : Pattern to search for.

**Return:**
- Start index of the first match; `0` if pattern is empty; `stv_npos` if not found or text too short.

### stv_sundaySearch
```c
size_t stv_sundaySearch(strview stv_text, strview stv_pat);
```
Sunday algorithm substring search, generally faster for longer patterns.

**Parameters:**
- `stv_text` : Text to search in.
- `stv_pat` : Pattern to search for.

**Return:**
- Start index of the first match; `0` if pattern is empty; `stv_npos` otherwise.

## First/Last Index Queries

### Generic macros

#### stv_firstIndex
```c
#define stv_firstIndex(stv, target, invert) _Generic((target), char: stv_firstChar, strview: stv_firstCharset, stv_charClassFn: stv_firstCharClass)((stv), (target), (invert))
```
Dispatches to the appropriate first-index function based on the type of `target`. If `invert` is `true`, searches for the first character **not** matching.

#### stv_lastIndex
```c
#define stv_lastIndex(stv, target, invert) _Generic((target), char: stv_lastChar, strview: stv_lastCharset, stv_charClassFn: stv_lastCharClass)((stv), (target), (invert))
```
Dispatches to the appropriate last-index function. Inverts the condition if `invert` is `true`.

### Character search

#### stv_firstChar
```c
size_t stv_firstChar(strview stv, const char ch, bool invert);
```
Find the first occurrence of `ch` (or first not equal if `invert`).

**Parameters:**
- `stv` : View to search.
- `ch` : Character to locate.
- `invert` : If `true`, search for the first character **not** equal to `ch`.

**Return:**
- Index of the match, or `stv_npos` if not found or view empty.

#### stv_lastChar
```c
size_t stv_lastChar(strview stv, const char ch, bool invert);
```
Find the last occurrence of `ch` (or last not equal if `invert`).

**Parameters:**
- `stv` : View to search.
- `ch` : Character to locate.
- `invert` : If `true`, search for the last character **not** equal to `ch`.

**Return:**
- Index of the match, or `stv_npos` if not found or view empty.

### Charset search

#### stv_firstCharset
```c
size_t stv_firstCharset(strview stv, strview charset, bool invert);
```
Find the first character that is (or is not, if `invert`) a member of `charset`.

**Parameters:**
- `stv` : View to search.
- `charset` : Character set view. If empty, always returns `stv_npos`.
- `invert` : If `true`, search for the first character **not** in the charset.

**Return:**
- Index of the match, or `stv_npos`.

#### stv_lastCharset
```c
size_t stv_lastCharset(strview stv, strview charset, bool invert);
```
Find the last character that is (or is not) a member of `charset`.

**Parameters:**
- `stv` : View to search.
- `charset` : Character set view.
- `invert` : If `true`, search for the last character **not** in the charset.

**Return:**
- Index of the match, or `stv_npos`.

### Classification-function search

#### stv_firstCharClass
```c
size_t stv_firstCharClass(strview stv, stv_charClassFn handle, bool invert);
```
Find the first character that satisfies `handle` (or does not satisfy if `invert`).

**Parameters:**
- `stv` : View to search.
- `handle` : Classification function. If `NULL` or view empty, returns `stv_npos`.
- `invert` : If `true`, search for the first character that does **not** satisfy the class.

**Return:**
- Index of the match, or `stv_npos`.

#### stv_lastCharClass
```c
size_t stv_lastCharClass(strview stv, stv_charClassFn handle, bool invert);
```
Find the last character that satisfies `handle` (or does not satisfy).

**Parameters:**
- `stv` : View to search.
- `handle` : Classification function.
- `invert` : If `true`, search for the last character that does **not** satisfy.

**Return:**
- Index of the match, or `stv_npos`.

## Difference Detection

### stv_firstDiff
```c
size_t stv_firstDiff(strview stv_left, strview stv_right);
```
Find the first byte position where two views differ (from left to right).

**Parameters:**
- `stv_left` : Left view.
- `stv_right` : Right view.

**Return:**
- Index of first difference, or `stv_npos` if identical.

**Details:**
- If one is a prefix of the other, returns the length of the shorter view.
- If one is empty and the other is not, returns `0`.

### stv_lastDiff
```c
size_t stv_lastDiff(strview stv_left, strview stv_right);
```
Find the last byte position where two views differ (from right to left).

**Parameters:**
- `stv_left` : Left view.
- `stv_right` : Right view.

**Return:**
- Index of last difference (from the start), or `stv_npos` if identical.

## Predicates

### Generic macros

#### stv_every
```c
#define stv_every(stv, target) _Generic((target), char: stv_everyChar, stv_charClassFn: stv_everyIf)((stv), (target))
```
Dispatches to `stv_everyChar` or `stv_everyIf`. Returns `true` if all characters satisfy the condition.

## Number Parsing

### stv_ch2digit
```c
int stv_ch2digit(char ch);
```
Convert a character to its numeric digit value (0‑35).

**Parameters:**
- `ch` : Input character.

**Return:**
- Digit value for `'0'`‑`'9'` (0‑9), `'A'`‑`'Z'` (10‑35), or `'a'`‑`'z'` (10‑35).  
  Returns `-1` if the character is not a valid digit.

### stv_parseIntBase
```c
int stv_parseIntBase(strview stv, strview* remaining);
```
Detect the numeric base from a view's prefix. Supports `0b`/`0B` (binary), `0o`/`0O` (octal), `0d`/`0D` (decimal), and `0x`/`0X` (hex).  
A leading `'0'` without a recognised prefix is treated as decimal and the `'0'` remains as part of the number.  
If the view is empty, returns `0` and sets `*remaining` to the empty view.

**Parameters:**
- `stv` : View to examine.
- `remaining` : Optional output pointer to receive the part after the prefix. If `NULL`, the remainder is discarded.

**Return:**
- The detected base (2, 8, 10, or 16). Returns `0` if the view is empty.

**Details:**
- Unlike the C standard library, a lone `'0'` does **not** imply octal. Only explicit `0o`/`0O` triggers octal.  
- When a prefix is recognised, the returned `remaining` starts after the two‑character prefix; otherwise it covers the original view.

### stv_parseInum
```c
intmax_t stv_parseInum(strview stv, int base, strview* remaining);
```
Parse a signed integer from a string view.  
Leading whitespace (as defined by [`stv_whitespace`](#stv_whitespace)) is skipped, then an optional `'+'` or `'-'` sign is consumed. Digits in the given base are parsed until a non‑digit or the end of the view.  
If `base` is `0`, it is auto‑detected using [`stv_parseIntBase`](#stv_parseintbase). The base must be in the range 2‑36; otherwise the function returns `0` without consuming additional characters beyond whitespace and sign.

**Parameters:**
- `stv` : View to parse.
- `base` : Numeric base (2‑36), or `0` for auto‑detection.
- `remaining` : Optional output pointer that receives the first unprocessed character. If `NULL`, the remainder is discarded.

**Return:**
- The parsed value as `intmax_t`. Returns `0` if no digits were found or the base is invalid.  
  On overflow, returns `INTMAX_MAX` for positive or `INTMAX_MIN` for negative, and `remaining` points past the digits.

### stv_parseUnum
```c
uintmax_t stv_parseUnum(strview stv, int base, strview* remaining);
```
Parse an unsigned integer from a string view.  
Works identically to [`stv_parseInum`](#stv_parseinum) except that the result is `uintmax_t`. Negative numbers are converted by interpreting the absolute value in the unsigned domain (e.g., `"-40"` yields `UINTMAX_MAX - 39`).

**Parameters:**
- `stv` : View to parse.
- `base` : Numeric base (2‑36), or `0` for auto‑detection.
- `remaining` : Optional output pointer for the remainder.

**Return:**
- The parsed value as `uintmax_t`. Returns `0` if no digits or invalid base.  
  On overflow, returns `UINTMAX_MAX`.
